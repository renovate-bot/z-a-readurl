# Install

Simply load like a regular plugin, i.e.:

```zsh
zi light z-shell/z-a-readurl
```

After executing the above command it's then possible to use the `dlink''` and `dlink0''` ices and also the special `as'readurl|‚Ä¶'` value of the `as''` ice.
